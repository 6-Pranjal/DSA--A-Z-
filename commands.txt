# Step 1: Switch to main branch
git checkout main

# Step 2: Pull latest changes (important)
git pull origin main

# Step 3: Merge dev into main
git merge dev

# Step 4: Push to remote main
git push origin main


git push origin dev:main
👉 This directly pushes dev → main without switching branches.



Always pull latest main before merging to avoid conflicts
If conflicts happen → Git will ask you to resolve them manually
Don’t use force push unless necessary:

git push --force origin main   # risky ❌
